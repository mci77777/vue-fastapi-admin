🔐 JWT Token 注入工具

自动将 JWT token 注入到浏览器 localStorage

步骤 1: 获取 JWT Token

步骤 2: 注入到 localStorage

步骤 3: 测试 WebSocket

📖 使用指南

1
自动登录：点击 "自动登录并获取 Token" 按钮，系统会自动调用后端 API 获取测试 token
2
注入 Token：点击 "注入 Token" 按钮，将 token 保存到 localStorage.ACCESS_TOKEN
3
测试连接：点击 "测试 WebSocket 连接" 按钮，验证 JWT 认证是否成功
💡 提示：此工具生成的是 测试 Token（HS256 算法），用于本地开发测试。生产环境应使用 Supabase Auth 签发的真实 token（ES256 算法）。
